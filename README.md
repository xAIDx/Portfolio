# My Portfolio
Welcome to my personal website project. This is a simple space where I have organized my current work in web development and design.

### Features:
* **UI:** A clean, dark theme focused on content.
* **Interactivity:** Dynamic skill charts and a basic copy-to-clipboard system for contacts.
* **Technology:** Built using a standard stack without external frameworks.

### Tech Stack:
* HTML5
* CSS3
* Vanilla JavaScript
